# 塔罗牌AI解牌系统 — Coding Plan

## 项目概述
构建一个线上塔罗牌系统，支持从抽牌到AI解牌的全流程体验。

## 核心功能
1. **牌组展示** — 78张塔罗牌（22张大阿尔卡纳 + 56张小阿尔卡纳），支持正逆位
2. **多种牌阵** — 单张牌、三张牌（过去/现在/未来）、凯尔特十字等
3. **AI解牌** — 基于抽牌结果，提供深度AI解读
4. **历史记录** — 保存占卜记录，支持回顾
5. **动画效果** — 抽牌、翻牌、牌阵排列等流畅动画

## 技术栈
- **前端**: React + TypeScript + Tailwind CSS + shadcn/ui
- **动画**: Framer Motion
- **状态管理**: Zustand
- **AI解牌**: 模拟AI解牌（展示架构，预留AI接口）
- **部署**: 静态部署

## 设计风格
- 神秘、优雅、低饱和度暗色系
- 星空/宇宙元素背景
- 金色调点缀
- 卡片式布局
